## run

```
 1. // 编译ffmpeg 静态库
  sudo sh scripts/compile_libx.sh

 2. make // 修改Makefile, 指定target 生成可执行文件

 3. ./demo _media/xx.ts xxx

 4. 启动服务 sever/rgb.html等 可查看解码视频帧的rgb图像、可播放解码音频的pcm数据

```

```
  先注释掉 base.h中 #define MAIN_RUN

  1. // 编译用于webassembly 的ffmpeg 静态库
```
